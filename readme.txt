Audio player plugin for Radiojar platform.
Insert with shortcode [rj-player].

== Description ==

<strong>Important Note: </strong>
This plugin is only functional for stations operating under Radiojar.
</br>
It's the simplest way to add your Radiojar station's player to your WordPress site using shortcode [rj-player]</br>
Fill in your station’s stream name, select the player of your preference and have the player into your site or post easily. </br>
An advanced selection for Ajaxify is also provided, for uninterrupted playback during the user's navigation in the site.
</br>
For details on Ajaxify, kindly refer to  <a target="_blank" href="https://wordpress.org/plugins/ajaxify-wordpress-site/">Ajaxify WordPress Site(AWS)</a>.

== Radiojar Audio Player in 3 steps: ==
<br />1: Install Radiojar Audio Player plug-in.
<br />2: Fill in the stream name.
<br />3: Add the player to any post/page using shortcode [rj-player].

== Installation ==

1. Upload `radiojar-audio-player` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the Radiojar player menu that appears in your admin menu.
4. Add the player to any post/page using shortcode [rj-player].
